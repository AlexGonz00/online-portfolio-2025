VR Construction Management

An educational VR application

Through the summer of 2024, me and a team of construction science experts developed an educational mixed reality Unity application that simulates construction site management tasks. This application is tailored for a secondary educational audience, using 3D UI design elements to help facilitate user engagement.

This simulation explores various construction site activities and the sensor technologies associated with them. The project also includes a visual code block system to teach students how they can optimize construction tasks for improved safety, efficiency, and cost-effectiveness.
